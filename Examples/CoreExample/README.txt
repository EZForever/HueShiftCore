How to create a new Core project:

1. Create a new C++ empty project under this solution
2. Change project configuration type to "Dynamic Library (.dll)"
3. Add reference to CoreCommon, DetoursCore and Detours projects
4. Copy config.h and hook.cpp from this project to your new project
5. Edit config.h to your liking
6. Edit hook.cpp to add your hook routines
7. Build project and enjoy
